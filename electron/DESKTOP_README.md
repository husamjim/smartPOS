# دليل نسخة سطح المكتب (Desktop README)

هذا المجلد يحتوي على إعدادات تشغيل وتغليف نظام **smart POS** كبرنامج مستقل لسطح المكتب يعمل بالكامل محلياً (Offline) على أنظمة Windows/macOS/Linux.

## 🌟 مميزات نسخة سطح المكتب
1. **العمل بدون إنترنت (100% Offline)**: يعتمد النظام بالكامل على قاعدة بيانات IndexedDB المحلية (عبر Dexie.js) لحفظ المنتجات، المبيعات، الموردين، والعملاء.
2. **برنامج تثبيت مستقل (.exe Installer)**: يتم تجميع الكود لإنتاج ملف تثبيت معالج NSIS يقوم بتنزيل البرنامج وإنشاء اختصارات على سطح المكتب وقائمة Start.
3. **أمان معزز**: إيقاف تفعيل `nodeIntegration` وتفعيل `contextIsolation` لمنع أي ثغرات حقن برمجية (XSS) من الوصول إلى موارد نظام التشغيل.
4. **دعم مباشر للأجهزة الملحقة**: التوصيل المتكامل بالطابعات الحرارية عبر WebUSB وموازين الوزن الإلكترونية عبر مخرجات Serial.

## 🛠️ متطلبات البناء والتجميع
* تثبيت Node.js (الإصدار 18 أو أحدث) من [https://nodejs.org](https://nodejs.org).

## 🚀 كيفية بناء نسخة التثبيت للويندوز (Production Build)
من المجلد الرئيسي للمشروع، قم بتشغيل الأوامر التالية:

```bash
# 1. تثبيت الحزم البرمجية
npm install

# 2. بناء واجهة المستخدم الأمامية
npm run build:frontend

# 3. تجميع برنامج التثبيت (Installer) للويندوز
npm run build:all
```

بعد انتهاء عملية التجميع، ستجد ملف التثبيت المكتمل (`smart POS Setup 1.0.0.exe`) داخل مجلد:
`dist-electron/`

## 🖥️ تشغيل التطبيق محلياً في وضع التطوير (Quick Launch)
```bash
# بناء الواجهة مرة واحدة
npm run build:frontend

# تشغيل البرنامج مباشرة
npm start
```
